## 网页图片大师 — 摸鱼神器

> **永久免费 · 本地运行 · 开源 · MIT 协议**

一个优雅的网页图片管理工具，让你在浏览网页时自由控制图片的显示与隐藏，支持多种交互方式和丰富的统计面板，是工作间隙"摸鱼"的绝佳伴侣。

---

### 一、核心功能总览

#### 1. 图片显隐切换（基础功能）
- **一键隐藏/显示**：点击仪表盘或图片旁的浮动按钮，即可将页面所有图片（`<img>`、`<svg>`、背景图）模糊隐藏或恢复显示
- **单图独立控制**：每张图片左上角出现「显/隐」浮动按钮，可单独控制该图片
- **防误触设计**：静止触发机制，避免鼠标偶然划过时误操作
- **优雅过渡动画**：隐藏/显示带有 0.5s 的模糊+透明度渐变动画（可关闭）

#### 2. 图片放大查看（四种模式）
| 模式 | 触发方式 | 特点 |
|------|---------|------|
| **关闭** | — | 禁用放大功能 |
| **悬停放大** | 鼠标悬停 2 秒 | 自动放大，可设置"离开图片即关闭"或"仅靠按钮关闭" |
| **中键放大** | 鼠标中键点击 | 快速放大图片，中键再次点击关闭 |
| **点击放大** | 点击「放」按钮 | 最稳妥的手动方式 |

- **中心缩放动画**：放大时图片从原位置平滑飞入屏幕中央，带阴影和背景暗化效果
- **滚轮缩放**：放大后可用滚轮继续缩放图片大小
- **拖拽移动**：放大后的图片可自由拖拽到任意位置

#### 3. 钉图模式（多图共存）
- 开启后，放大图片不会关闭之前的图片，支持**同时查看多张图片**
- 每张钉住的图片有独立的关闭按钮（×）
- 支持滚轮缩放和拖拽排列，适合对比多张图片

#### 4. 阅兵模式（图片阵列）
- **一键阵列**：将所有页面图片提取出来，以整齐的网格阵列展示
- **飞入动画**：图片从原位置平滑飞入阵列位置，带缓动效果
- **独立交互**：阵列中的每张图可单独滚轮缩放、拖拽移动
- **中键还原**：中键点击单图，该图会飞回原网页位置并恢复隐藏状态
- **尺寸过滤**：可设置最小/最大宽高过滤条件，只展示符合尺寸的图片
- **点击背景退出**：点击空白处或关闭按钮，所有图片飞回原位

#### 5. 虚线辅助框
- 隐藏的图片位置显示**动态虚线边框**（跑马灯效果）
- 可设置"始终显示"或"仅悬停时显示"
- 帮助定位被隐藏图片的原始位置

#### 6. 悬停预览
- 开启后，鼠标悬停在隐藏图片上时，临时显示原图（不放大）
- 移开鼠标后自动恢复隐藏
- 适合快速查看图片内容而不完全显示

### 二、统计面板（仪表盘）

固定在屏幕边缘的透明玻璃态面板，实时显示多种数据：

#### 基础信息
- **时间**（时:分 / 时:分:秒）
- **当前域名**

#### 图片统计
- **隐藏数量** / **图片总数** / **隐藏占比**
- **疑似广告数量**（自动检测 iframe 和图片广告）

#### 性能监控
- **内存占用**（JS Heap）
- **页面加载耗时**
- **资源请求数**

#### 用户行为追踪（支持"本次会话" / "今日"切换）
- **鼠标移动距离**（自动换算为米/公里）
- **点击次数**
- **滚动距离**
- **阅读字数**（根据页面文字密度估算）

### 三、界面与交互

#### 仪表盘特性
- **拖拽定位**：可自由拖动到屏幕任意位置
- **自动贴边**：靠近左右边缘时自动吸附，变成圆角半贴边样式
- **自动休眠**：静止时自动收缩为小圆点（带呼吸灯效果），鼠标移入展开
- **透明背景**：休眠后可完全透明，不遮挡内容

#### 菜单系统
- **三级菜单结构**：主菜单 → 设置 → 显示内容
- **智能朝向**：菜单自动判断屏幕空间，向左侧或右侧展开，避免超出屏幕
- **快捷键**：`Alt + H` 快速显示/隐藏面板

#### 动画控制
- 支持**一键禁用所有动画**，获得最流畅的体验

---

### 四、数据与配置

#### 持久化存储
- **网站级配置**：每个网站的隐藏状态、过滤条件独立记忆
- **全局配置**：统计项选择、放大模式等跨网站同步
- **永久隐藏网站**：可将特定网站加入黑名单，面板默认不显示（仍可用 Alt+H 唤出）
- **全局统计**：今日鼠标距离、点击次数等数据跨页面累计

#### 隐私说明
- 所有数据**本地存储**，不上传任何服务器
- 纯前端运行，无需网络权限

### 五、使用教程

#### 快速上手
1. **安装脚本**：通过 Tampermonkey / Greasemonkey 等扩展安装
2. **打开网页**：脚本自动在右下角出现仪表盘（绿色灯 = 图片显示，红色灯 = 图片隐藏）
3. **点击仪表盘**：切换全页图片的隐藏/显示状态
4. **悬停图片**：出现「显/隐」和「放」按钮，进行单图操作

#### 进阶操作
| 操作 | 方法 |
|------|------|
| 单图隐藏/显示 | 悬停图片 → 点击「显」或「隐」按钮 |
| 快速放大 | 中键点击图片（中键模式）或悬停 2 秒（悬停模式） |
| 多图对比 | 设置中开启「钉图模式」→ 多次放大图片 |
| 查看所有图片 | 菜单 → 「开启阅兵模式」 |
| 过滤小图标 | 设置 → 阅兵过滤 → 设置最小宽高 |
| 关闭动画 | 设置 → 禁用动画 |
| 唤出隐藏面板 | `Alt + H` |

#### 推荐设置组合
- **轻度摸鱼**：悬停放大 + 虚线辅助 + 自动休眠
- **深度摸鱼**：中键放大 + 钉图模式 + 阅兵阵列
- **极简模式**：关闭放大 + 禁用动画 + 仅显示隐藏数量

### 六、技术亮点

- **零依赖**：纯原生 JavaScript，无外部库
- **高性能**：MutationObserver 智能监听 DOM 变化，500ms 轮询优化
- **防屏蔽**：监听自身 DOM 被移除，自动恢复，对抗页面清理
- **精确坐标**：考虑滚动、边框、定位等多种因素计算图片位置
- **流畅动画**：全部使用 CSS3 硬件加速过渡，requestAnimationFrame 优化面板刷新
